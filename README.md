# Centralized Dataset Testing

This repository contains the Jupyter notebook used to perform the centralized machine learning experiments for the paper *AI-Powered Blockchain for Privacy-Preserving Federated Learning in Healthcare Systems*.

The notebook includes:
- Data preprocessing and feature engineering
- Centralized MLP model training and evaluation
- Centralized XGBoost model training and evaluation
- Performance metrics including accuracy, precision, recall, F1-score, and AUC-ROC

Dataset:
- Diabetes 130-US Hospitals for Years 1999–2008 (UCI Machine Learning Repository)
